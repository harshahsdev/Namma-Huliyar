# 🌐 Namma Huliyar – Local Business Directory App

Namma Huliyar is a full-stack MERN application that helps users discover local businesses, view details, read and write reviews, and get directions using Google Maps.

---

## 🚀 Features

- 🔍 Search businesses by name or category  
- 🏪 View business details (name, location, description)  
- ⭐ Ratings and reviews system  
- ✍️ Add reviews (authenticated users)  
- 📍 Get directions via Google Maps  
- 🔐 User authentication (JWT-based login/signup)  

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Axios

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Mongoose)
